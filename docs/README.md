# Welcome

## **Welcome to BitLoan Documentation**

Welcome to the official BitLoan documentation.\
This guide will give you a complete overview of how the BitLoan protocol works—from high-level concepts to deep technical architecture. Whether you’re a borrower, liquidity supplier, liquidator, or developer, you’ll find everything you need to understand, integrate, and build on the protocol.

Here, you’ll explore how BitLoan enables secure, permissionless lending using **ETH as collateral** to borrow **cNGN or other ERC-20 assets**, powered entirely by transparent smart contracts.

Use this documentation to:

* Learn the fundamentals of the protocol
* Understand how collateral, borrowing, interest, and liquidation work
* Explore the contract architecture
* Review the economic and risk parameters
* Integrate BitLoan into your application
* Join the ecosystem as a user or developer
